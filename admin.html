🔑 Acesso ao GitHub

Como gerar a chave de acesso?

1. Acesse github.com/settings/personal-access-tokens/new
2. Em "Repository access", escolha "Only select repositories" e selecione o seu repositório do site
3. Em "Permissions → Repository permissions", ache "Contents" e mude pra "Read and write"
4. Clique em "Generate token" e copie o código gerado (começa com github_pat_)
5. Cole no campo acima
